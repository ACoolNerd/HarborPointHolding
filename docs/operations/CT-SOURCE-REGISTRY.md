# Connecticut Source Registry

| Source | Primary use | Automation rule |
|---|---|---|
| CT statewide Parcel and CAMA | Parcel geometry and assessor attributes | Permitted public data; preserve vintage and municipality limitations |
| CT Secretary of the State business search | Entity verification | Use official access; do not treat entity match as title proof |
| CT Secretary of the State lien search | Filed lien discovery | Verify record identity and date |
| Municipal land records | Deeds, mortgages, assignments, releases, lis pendens | Respect each portal's terms; require manual verification when needed |
| Municipal assessor and GIS | Parcel, mailing, use, and assessment research | Research evidence only |
| Municipal building and zoning | Permits, applications, approvals, and use questions | Do not infer final approvals |
| Connecticut Judicial Branch lookup | Case discovery | Use permitted access; no pressure targeting |
| CT DEEP GIS | Environmental and resource screening | Preliminary screening only |
| CT DECD | Town, construction, housing, and economic context | Preserve publication date |
| FEMA NFHL | Flood-hazard screening | Confirm against current official products |

Licensed MLS, CoStar, LoopNet, Crexi, Regrid, skip-tracing, and similar data may be used only through authorized accounts, APIs, exports, or contracts. Never automate around paywalls, login controls, rate limits, robots restrictions, or license terms.

Every evidence record stores publisher, URL, dataset or record date, retrieval timestamp, permitted-use note, verification status, and any conflict.
