# KassConnect / Kassandra Brand Guide

## Brand position

Prepared. Strategic. Local. Commercial-capable. Human.

## Public brand line

**Kassandra Gonzales, REALTOR® — Connecticut Property Strategy powered by KassConnect**

## Core promise

Move with clarity, confidence, and strategy.

## Visual style

- Primary: black, white, warm gray
- Accent: champagne gold
- Optional support: deep slate, soft cream
- Avoid loud investor-wholesaler graphics
- Avoid distressed-sale pressure visuals

## Typography

- Heading: Playfair Display, Cormorant Garamond, or Georgia
- Body: Inter, Montserrat, Arial, or system sans-serif

## Tone

- Calm
- Professional
- Direct
- Client-first
- Clear about options and disclosures
- Never predatory or fear-based
