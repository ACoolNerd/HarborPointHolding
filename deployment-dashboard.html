🚀 Quick Deploy

Local (Docker):

docker compose up

Export Static Site:

node deploy.js export

Deploy to GitHub Pages:

git push origin main

GitHub Actions will automatically deploy on push.

Deploy to FTP/VPS:

node deploy.js ftp --env=production

Requires FTP credentials in .env.local

⚙️ Environment Setup

1. Copy the example config:

cp .env.example .env.local

2. Add your deployment credentials to .env.local:

  • FTP_HOST, FTP_USER, FTP_PASS (for shared hosting)
  • HUBSPOT_PORTAL_ID, HUBSPOT_FORM_GUID (for CRM)
  • CALENDLY_URL (for scheduling link)

3. For GitHub Actions:

Go to: GitHub repo → Settings → Secrets and variables → Actions

Add repository secrets for sensitive data (GitHub Actions uses these automatically).
