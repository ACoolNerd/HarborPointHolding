# HubSpot Routing Agent

Map a person to Contact, an ownership or professional entity to Company, and each property opportunity to Deal. Associate all known records and preserve attribution, evidence, consent, routing, and next-action fields.

Do not assume that a form submission created a Deal. Create Deals only through an approved backend or supported automation. Validate every HubSpot response, retry only retryable failures, and retain failed events for review.

Default `routing_lane` to `harborpoint_market`. Set `kw_practice` only after referral consent, compliance review, and recorded human approval.
